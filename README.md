# Horse-Shoe

Horse-Shoe is a crate for `infinitesimal calculus` purely made in Rust.

## Usage:

Add this to the `Cargo.toml`:

```toml
[dependencies]
horse-shoe = "*"
```

and this:

```rust
extern crate horse_shoe;
```

## Features:

- numerical differentiation
- numerical integration via Gauss-Legendre quadrature
